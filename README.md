Lernspiel Requirements Engineering

Ein interaktives Lernspiel, das die Grundlagen des Requirements Engineering vermittelt. Spieler:innen lernen, wie man Anforderungen von Stakeholdern erhebt, dokumentiert und in konkrete Umsetzungen überträgt, dargestellt am Beispiel einer Sandwichbestellung.

Features

Elicitation Phase – Notizen von Stakeholder-Wünschen sammeln.

Documentation Phase – Anforderungen aus Notizen formulieren.

Development Phase – Anforderungen in ein „Sandwich“ umsetzen.

Results Page – Feedback zur Genauigkeit der Umsetzung und Anforderungen.

Interaktive Maskottchen-Erklärungen – Lerninhalte mit kurzen Tipps und Erklärungen.

Dynamische Fehleranalyse – zeigt Abweichungen von „perfekten“ Sandwichen und fehlende Anforderungen.

Gamification – Punkte/Fehleranalyse als spielerisches Feedback.
